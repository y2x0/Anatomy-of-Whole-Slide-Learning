# Hybrid Composition And Operator Noncommutativity

## 1. Two operators, two meanings

Let H be a matrix of instance states and G the geometry or support object. A
context operator produces contextual states:

```math
\widetilde H
=
\mathcal C_{\theta}(H,G).
```

A readout operator maps contextual states to a slide statistic:

```math
z
=
\mathcal R_{\phi}(\widetilde H,G).
```

A hybrid MIL predictor is therefore

```math
F_{\theta,\phi}(H,G)
=
\mathcal R_{\phi}
\left(
\mathcal C_{\theta}(H,G),G
\right).
```

The context operator changes the representation of each instance. The readout
operator determines which representation survives.

## 2. Why order matters

Let M be a linear mean readout and A a graph propagation matrix. The two
possible compositions are

```math
F_1(H)=M(AH),
\qquad
F_2(H)=A(MH).
```

The second expression may not even be dimensionally valid because M removes the
instance axis before graph propagation. Even when both are defined, they are
not equal unless the graph operator and readout satisfy an intertwining relation.

For a parent assignment matrix P, graph context before coarsening is

```math
P^{\mathsf T}AH,
```

whereas coarsening before a parent graph operator A_P is

```math
A_P P^{\mathsf T}H.
```

Equality requires

```math
P^{\mathsf T}A=A_P P^{\mathsf T}.
```

This is a strong compatibility condition. A coarse graph is not automatically
the quotient of a fine graph.

## 3. Attention changes the operator algebra

For attention readout

```math
R_{\mathrm{att}}(H)
=
\sum_{j=1}^{n}\alpha_j(H)h_j,
\qquad
\alpha_j(H)
=
\frac{\exp s_{\phi}(h_j)}
{\sum_{k=1}^{n}\exp s_{\phi}(h_k)},
```

the readout is nonlinear in H. A contextualized attention readout is

```math
R_{\mathrm{att}}(AH)
=
\sum_j
\alpha_j(AH)(AH)_j.
```

The attention weights now depend on neighborhoods. A high-weight instance is
not necessarily locally positive; it may be a graph context that summarizes
positive neighbors.

For a class-specific readout with logit w_c^{\mathsf T}z, the effective
fine-level contribution is no longer just alpha_j w_c^{\mathsf T}h_j because
the score alpha_j depends on all contextual states.

## 4. Geometry determines what a hybrid can express

For a flat set, a permutation-equivariant C followed by invariant R gives an
invariant bag function. For a graph, equivariance is with respect to simultaneous
node and adjacency permutations:

```math
\mathcal C(QH,QAQ^{\mathsf T})
=
Q\mathcal C(H,A),
\qquad
\mathcal R(QH,QAQ^{\mathsf T})
=
\mathcal R(H,A).
```

A hybrid may therefore be set-invariant while remaining geometry-sensitive:
permuting nodes together with their coordinates leaves the output unchanged,
but changing the graph support changes the output.

## 5. Tensorized hybrid map

With multiple context operators, write

```math
H^{(0)}=H,
\qquad
H^{(r+1)}
=
\mathcal C_r
\left(
H^{(r)},G^{(r)}
\right),
\qquad
z=
\mathcal R
\left(
H^{(R)},G^{(R)}
\right).
```

The model's inductive bias is the ordered tuple

```math
\left(
\mathcal C_0,\ldots,\mathcal C_{R-1},
\mathcal R,
G^{(0)},\ldots,G^{(R)}
\right).
```

Calling this model attention MIL or graph MIL alone hides the fact that the
readout may be attention while the information being weighted was generated by
graph, transformer, hierarchy, or state-space context.

## 6. A useful diagnostic

For any proposed hybrid, write both legal orderings and ask:

`text
- Does the other ordering have a valid shape?
- If valid, does it satisfy an intertwining or commutation relation?
- Which operator first removes the fine-instance axis?
- Does the final head see coordinates, adjacency, parent maps, or ordering?
- Is the interpretation based on pre-context features or contextual features?
`

These questions separate a genuinely new inductive bias from a new name for a
known composition.
`

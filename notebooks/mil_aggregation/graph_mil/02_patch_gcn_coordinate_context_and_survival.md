# Patch-GCN Coordinate Context And Survival

Source:

```text
Chen et al., Whole Slide Images are 2D Point Clouds: Context-Aware Survival
Prediction using Patch-based Graph Convolutional Networks, MICCAI 2021.
https://arxiv.org/abs/2107.13048
```

## 1. Graph Construction

For WSI `j` of patient `i`, the patch encoder produces:

```math
h_{ijv}^{(0)}
=
f_{\mathrm{ResNet}}(x_{ijv})
\in
\mathbb{R}^{1024},
```

and tissue segmentation provides a coordinate:

```math
c_{ijv}
\in
\mathbb{R}^{2}.
```

The WSI graph is:

```math
G_{ij}
=
\left(
V_{ij},E_{ij}^{\mathrm{coord}},X_{ij}
\right),
```

where the support is induced by coordinate neighborhoods. In a directed
notation:

```math
u\in\mathcal{N}_{ij}(v)
\Longleftrightarrow
u\in\mathrm{kNN}_{8}(c_{ijv}).
```

The central geometry is therefore spatial coordinate kNN, not feature-space
similarity. The node matrix is:

```math
H_{ij}^{(0)}
\in
\mathbb{R}^{M_{ij}\times 1024}.
```

## 2. Local Context Operator

Patch-GCN adapts a DeepGCN-style graph layer. For an edge from `u` to `v`, a
paper-faithful generic message is:

```math
m_{ij,vu}^{(\ell)}
=
\mathrm{ReLU}
\left(
h_{ij,u}^{(\ell)}
+
\mathbf{1}\left(h_{ij,e}^{(\ell)}\right)
h_{ij,e}^{(\ell)}
\right)
+
10^{-7}.
```

The WSI construction is primarily defined by coordinate support; the general
edge feature term should not be read as evidence that the WSI graph uses a
biological edge attribute absent from the construction.

The paper's softmax aggregation is written featurewise. For feature coordinate
`r`:

```math
a_{ij,vu,r}^{(\ell)}
=
\frac{
\exp\left(m_{ij,vu,r}^{(\ell)}\right)
}{
\sum_{s\in\mathcal{N}_{ij}(v)}
\exp\left(m_{ij,vs,r}^{(\ell)}\right)
},
```

```math
\bar m_{ij,v,r}^{(\ell)}
=
\sum_{u\in\mathcal{N}_{ij}(v)}
a_{ij,vu,r}^{(\ell)}
m_{ij,vu,r}^{(\ell)}.
```

The local statistic is thus a coordinatewise weighted first moment of the
neighbor messages. It is not one scalar ABMIL weight per patch.

The node update is:

```math
h_{ij,v}^{(\ell+1)}
=
\mathrm{MLP}_{\ell}
\left(
h_{ij,v}^{(\ell)}
+
\bar m_{ij,v}^{(\ell)}
\right).
```

The graph block uses residual and dense multi-depth connections. With four
graph convolution layers, a compact representation is:

```math
\widetilde h_{ij,v}
=
\mathrm{Concat}
\left(
h_{ij,v}^{(1)},
h_{ij,v}^{(2)},
h_{ij,v}^{(3)},
h_{ij,v}^{(4)}
\right).
```

The final node is therefore a multiscale contextual state, not only a
four-hop-smoothed copy of the input.

## 3. Global Attention Readout

For one WSI, let the contextualized node matrix be:

```math
\widetilde H_{ij}
=
\begin{bmatrix}
\widetilde h_{ij1}^{\top}\\
\vdots\\
\widetilde h_{ijM_{ij}}^{\top}
\end{bmatrix}
\in
\mathbb{R}^{M_{ij}\times d_{\mathrm{out}}}.
```

Global attention produces a scalar score and normalized weight:

```math
s_{ijv}
=
f_{\theta}\left(\widetilde h_{ijv}\right),
\qquad
b_{ijv}
=
\frac{\exp(s_{ijv})}
{\sum_{u=1}^{M_{ij}}\exp(s_{iju})}.
```

The WSI statistic is:

```math
z_{ij}
=
\sum_{v=1}^{M_{ij}}
b_{ijv}\widetilde h_{ijv}
\in
\mathbb{R}^{d_{\mathrm{out}}}.
```

This is a weighted first moment of graph-contextualized nodes. The two
weighting systems must be kept separate:

```text
local featurewise softmax:
    neighbor message -> contextualized patch

global scalar attention:
    contextualized patch -> WSI vector
```

## 4. Survival Head

For a single patient-level risk statistic:

```math
\eta_i
=
w^{\top}z_i+b.
```

If patient `i` has WSI subgraphs `G_{ij}`, the paper's object is naturally a
collection:

```math
\mathcal{G}_i
=
\left\{G_{ij}\right\}_{j=1}^{K_i}.
```

The safe abstract patient map is:

```math
\eta_i
=
\mathcal{H}_{\mathrm{surv}}
\left(
\mathcal{R}_{\mathrm{patient}}
\left(
\left\{z_{ij}\right\}_{j=1}^{K_i}
\right)
\right).
```

The exact across-WSI aggregation should not be invented when the paper does not
give it as a separate forward equation. What is explicit is that graph context
and global attention occur before patient-level survival supervision.

## 5. C/R/G/S And Failure

```text
C:
    DeepGCN-style coordinate-neighborhood context

R:
    global scalar attention, a weighted first moment of contextualized nodes

G:
    coordinate kNN over 2D patch locations

S:
    patient-level survival observation and Cox-style risk objective
```

The main collision is spatially important patches that are weak in isolation:
their state can become predictive only after nearby morphology has entered the
node through `C`. The main failure is a wrong coordinate graph: the readout can
only pool the contextual states generated by the supplied support.

# State-Space MIL Failure Modes and C/R/G/S

State-space MIL changes the slide object before the final aggregation. A WSI
patch collection begins as a multiset, but an SSM consumes an ordered sequence:

```math
\mathcal B_i
=
\left\{
h_{ij}:j=1,\ldots,L_i
\right\}
\longrightarrow
U_i^\sigma
=
\begin{bmatrix}
h_{i,\sigma(1)}^{\mathsf T}\\
\vdots\\
h_{i,\sigma(L_i)}^{\mathsf T}
\end{bmatrix}.
```

The complete predictor is:

```math
F_\theta
=
\mathcal H_\theta
\circ
\mathcal R_\theta
\circ
\mathcal C_{\theta}^{\mathrm{SSM}}
\circ
P_\sigma.
```

The central failure question is:

```math
\boxed{
\text{which order-dependent prefix statistics survive the finite state and the
final readout?}
}
```

The scan can be efficient and useful while still being a strong geometry
assumption. The analysis below separates order G, state-space context C,
readout R, and supervision S.

## 1. Source anchors

The paper-specific claims are anchored to:

| Method | Source note | Exact object used here |
| --- | --- | --- |
| S4MIL | [structured state-space readout](03_s4mil_structured_state_space_readout.md) | S4D-style sequence context and coordinatewise max |
| Mamba | [selective state spaces](04_selective_state_spaces_and_mamba_scan.md) | input-dependent selective recurrence |
| MambaMIL | [vanilla scan and MIL head](05_mambamil_vanilla_scan_and_mil_head.md) | stacked Mamba block and released attention readout |
| SR-Mamba | [sequence reordering operator](06_srmamba_sequence_reordering_operator.md) | original and segment-transposed scans |

The arguments below are mathematical consequences of these operators. They are
not claims that every implementation exhibits every failure.

## 2. Same bag, different prediction

Consider a scalar two-token bag and a final-state recurrence:

```math
s_t
=
a s_{t-1}+b u_t,
\qquad
s_0=0,
\qquad
z=s_2.
```

Order one-two gives:

```math
z_{12}
=
ab u_1+b u_2.
```

Order two-one gives:

```math
z_{21}
=
ab u_2+b u_1.
```

Their difference is:

```math
z_{12}-z_{21}
=
b(a-1)(u_1-u_2).
```

The map is order-sensitive unless a equals one, b equals zero, or the tokens
coincide. A final mean or max over token outputs does not generally remove this
dependence, because each token output was generated by a different prefix.

The unordered bag has one object but the SSM sees a family indexed by sigma:

```math
\left\{
U_i^\sigma:\sigma\in\mathfrak S_{L_i}
\right\}.
```

A single selected sigma is therefore part of G, not a harmless implementation
detail.

## 3. Order is an induced geometry

Let p-v be the physical coordinate of patch v and sigma-v its sequence position.
For a spatial adjacency set E-spatial, define order distortion:

```math
D_{\mathrm{ord}}(\sigma)
=
\frac{1}{|E_{\mathrm{spatial}}|}
\sum_{(u,v)\in E_{\mathrm{spatial}}}
\left|
\sigma(u)-\sigma(v)
\right|.
```

A large value means that physically adjacent patches are separated by many scan
steps. A one-way state can still carry their information, but it must preserve
it across a longer prefix.

The fraction of spatial neighbors within a scan window w is:

```math
Q_w(\sigma)
=
\frac{1}{|E_{\mathrm{spatial}}|}
\sum_{(u,v)\in E_{\mathrm{spatial}}}
\mathbf 1
\left\{
|\sigma(u)-\sigma(v)|\le w
\right\}.
```

Raster, Hilbert, random, learned, and file orders have different Q-w curves.
Reporting only sequence length hides this geometry.

For SR-Mamba, the segment-transposed order changes the distortion:

```math
D_{\mathrm{ord}}^{\mathrm{SR}}
=
D_{\mathrm{ord}}
\left(
\psi^{-1}\circ
\mathrm{transpose}_{R,N}
\circ
\mathrm{reshape}_{R,N}
\right).
```

The second scan supplies another structured geometry; it does not average over
all permutations.

## 4. Linear SSM memory kernel

A continuous state-space system is:

```math
\frac{dx(t)}{dt}
=
Ax(t)+Bu(t),
\qquad
y(t)=Cx(t)+Du(t).
```

Under zero-order hold with step Delta:

```math
\overline A
=
\exp(\Delta A),
```

```math
\overline B
=
\int_0^\Delta
\exp(\tau A)B\,d\tau.
```

The discrete recurrence is:

```math
x_t
=
\overline A x_{t-1}
+
\overline B u_t,
\qquad
y_t
=
Cx_t+Du_t.
```

With zero initial state:

```math
x_t
=
\sum_{k=1}^{t}
\overline A^{t-k}\overline B u_k.
```

Therefore:

```math
y_t
=
\sum_{k=1}^{t}
C\overline A^{t-k}\overline B u_k
+
Du_t.
```

The lag ell kernel is:

```math
K_\ell
=
C\overline A^\ell\overline B.
```

For a final-state readout, the contribution of token k is:

```math
\frac{\partial x_L}{\partial u_k}
=
\overline A^{L-k}\overline B.
```

The state does not store every token independently. It stores their images under
a finite collection of lag filters.

## 5. Finite-state collision

For fixed t, define the prefix map:

```math
\Psi_t:
\mathbb R^{td}
\longrightarrow
\mathbb R^N,
\qquad
\Psi_t(u_{1:t})=x_t.
```

If td is larger than N, the linear map cannot be injective. Its Jacobian rank
satisfies:

```math
\mathrm{rank}
\left(
\frac{\partial x_t}{\partial\mathrm{vec}(u_{1:t})}
\right)
\le N.
```

There are distinct prefixes with identical states:

```math
u_{1:t}\ne u_{1:t}',
\qquad
\Psi_t(u_{1:t})=\Psi_t(u_{1:t}').
```

For the linear system, a collision perturbation Delta satisfies:

```math
\sum_{k=1}^{t}
\overline A^{t-k}\overline B\Delta u_k
=
0.
```

Any downstream function that receives only x-t cannot recover that prefix
contrast. Nonlinear gates change the collision geometry but do not make the
finite state a lossless archive of an arbitrarily long WSI.

## 6. Stability and long-memory tradeoff

Let rho be the spectral radius of A-bar. If:

```math
\rho(\overline A)<1,
```

then stable modes decay. For a diagonalizable transition with eigenvalues
lambda-r:

```math
\left\|
C\overline A^\ell\overline B
\right\|
\lesssim
\kappa(U)
\max_r|\lambda_r|^\ell
\left\|C\right\|
\left\|\overline B\right\|.
```

Long-range influence requires eigenvalues near unit magnitude. But a mode near
one has a long effective memory and can make optimization sensitive to
initialization, normalization, and sequence length.

A simple memory half-life for one positive real mode is:

```math
\tau_{1/2}
=
\frac{\log(1/2)}{\log|\lambda|}.
```

If the WSI order places a related patch pair farther apart than this scale, the
linear mode attenuates their direct interaction.

The stable-memory tradeoff is not a theorem that long memory is impossible. It
is a constraint on a finite parameterized kernel.

## 7. Causal direction failure

For a causal linear scan:

```math
\frac{\partial y_t}{\partial u_k}
=
0
\qquad
\text{when }k>t.
```

A late token cannot change an earlier contextual output. If the readout uses all
outputs, the late token can still affect later outputs and the final prediction.
But a token's influence is direction-dependent.

For a final state:

```math
\frac{\partial x_L}{\partial u_k}
=
\overline A^{L-k}\overline B.
```

For a prefix output y-t, only k at most t contributes:

```math
\frac{\partial y_t}{\partial u_k}
=
\mathbf 1\{k\le t\}
C\overline A^{t-k}\overline B.
```

A bidirectional branch adds reverse-time paths, but it does not make an
arbitrary order equivalent to a set. SR-Mamba adds an additional structured
order, not every possible direction.

## 8. S4 convolution collisions

The structured convolution form is:

```math
Y_t
=
\sum_{k=1}^{t}
K_{t-k}U_k
+
DU_t.
```

Two sequences U and U-prime collide under a chosen output readout if:

```math
\sum_{k=1}^{t}
K_{t-k}
\left(
U_k-U_k'
\right)
+
D
\left(
U_t-U_t'
\right)
=
0
```

for every output t visible to R.

A max readout can create an even larger collision set:

```math
\max_t[Y_t]_r
=
\max_t[Y_t']_r
\qquad
\forall r.
```

The sequences can have different ordering, different number of high-response
tokens, and different extrema locations while producing the same slide vector.

## 9. Selective state-space failure

Mamba makes the effective transition input-dependent. A simplified selective
recurrence is:

```math
s_t
=
\overline A_t s_{t-1}
+
\overline B_t u_t,
```

where:

```math
\Delta_t
=
\mathrm{softplus}
\left(
W_\Delta u_t+b_\Delta
\right),
```

```math
B_t=W_Bu_t,
\qquad
C_t=W_Cu_t.
```

The transition can be written schematically as:

```math
\overline A_t
=
\exp(\Delta_t A).
```

The memory kernel is no longer a fixed function of lag. The influence of token
k on state t is a product:

```math
\frac{\partial s_t}{\partial u_k}
\approx
\left(
\prod_{r=k+1}^{t}\overline A_r
\right)
\frac{\partial\overline B_k u_k}{\partial u_k}
+
\text{gate derivatives}.
```

A high-response token can alter the future transition, not just add one value.
This gives selectivity but makes interpretation and stability input-dependent.

A perturbation at k changes downstream states through two paths:

```math
\frac{\partial s_t}{\partial u_k}
=
\underbrace{
\left(
\prod_{r=k+1}^{t}\overline A_r
\right)
\frac{\partial(\overline B_k u_k)}{\partial u_k}
}_{\text{value path}}
+
\underbrace{
\sum_{r=k}^{t}
\frac{\partial s_t}{\partial\overline A_r}
\frac{\partial\overline A_r}{\partial u_k}
+
\frac{\partial s_t}{\partial\overline B_r}
\frac{\partial\overline B_r}{\partial u_k}
+
\frac{\partial s_t}{\partial\overline C_r}
\frac{\partial\overline C_r}{\partial u_k}
}_{\text{selection path}}.
```

The attention-like selectivity is therefore not a probability of patch
positivity.

## 10. MambaMIL block and residual bypass

A MambaMIL block contains normalization, linear projections, convolution,
SiLU gates, a selective scan, and a residual:

```math
X'
=
\mathrm{Norm}(H^{(\ell)}),
```

```math
Y
=
\mathrm{SSM}
\left(
\mathrm{SiLU}
\left(
\mathrm{Conv1D}
\left(
\mathrm{Linear}(X')
\right)
\right)
\right),
```

```math
Z
=
\mathrm{SiLU}
\left(
\mathrm{Linear}(X')
\right),
\qquad
X''=Z\odot Y,
```

```math
H^{(\ell+1)}
=
H^{(\ell)}+X''.
```

The residual creates a direct tokenwise path. Thus the effective context is not
only the state memory:

```math
\frac{\partial H^{(\ell+1)}}{\partial H^{(\ell)}}
=
I
+
\frac{\partial X''}{\partial H^{(\ell)}}.
```

A long-range failure in the SSM branch can be partly masked by a local residual
path. Conversely, an apparent token-level explanation can reflect the residual
rather than the state transition.

## 11. Sequence reordering is a two-order ensemble

Let P-sigma be the original ordering and P-psi the SR-Mamba segment-transposed
ordering. The two branches compute:

```math
Y^{\mathrm{orig}}
=
\mathcal C_{\theta}
\left(
P_\sigma H
\right),
```

```math
Y^{\mathrm{reord}}
=
P_\psi^{-1}
\mathcal C_{\theta}
\left(
P_\psi H
\right).
```

The fused output is schematically:

```math
Y^{\mathrm{SR}}
=
\mathcal F
\left(
Y^{\mathrm{orig}},
Y^{\mathrm{reord}},
H
\right).
```

This is not permutation averaging:

```math
\mathcal C_{\mathrm{SR}}(H)
\ne
\frac{1}{|\mathfrak S_L|}
\sum_{\sigma\in\mathfrak S_L}
P_\sigma^{-1}
\mathcal C(P_\sigma H).
```

The model remains sensitive to which two orders were selected. The segment size
R changes the relation being exposed:

```math
L_{\mathrm{pad}}=RN,
\qquad
\mathsf X[r,n,:]
=
X_{\mathrm{pad}}[nR+r,:].
```

Small R emphasizes relationships across many segments at the same within-segment
offset. Large R makes the reordered path closer to the original path.

## 12. Padding as a graph and state artifact

For a slide of length L and padded length L-p, define:

```math
m_t
=
\mathbf 1\{t\le L\}.
```

An inert padded recurrence should satisfy:

```math
s_t
=
m_t
\left(
\overline A_t s_{t-1}
+
\overline B_tu_t
\right)
+
(1-m_t)s_{t-1}.
```

If zero padding is passed through a nonzero bias, convolution, normalization,
or selective gate, then:

```math
\mathcal C(U_{\mathrm{pad}})
\ne
\mathcal C(U)
\quad
\text{after de-padding}.
```

For attention readout, a padded token also changes the denominator unless its
score is masked:

```math
\alpha_t
=
\frac{\exp(s_t)m_t}
{\sum_{u=1}^{L_{\mathrm{pad}}}\exp(s_u)m_u}.
```

The mask must be applied to both state updates and readout support. De-padding
only the final output is insufficient.

## 13. Max readout failure

S4MIL uses coordinatewise max pooling:

```math
[z^{\mathrm{max}}]_r
=
\max_{1\le t\le L}[y_t]_r.
```

The maximizing location can differ by channel:

```math
t_r^\star
\in
\arg\max_{t}[y_t]_r.
```

Therefore z-max need not equal any actual token state. It preserves a
coordinatewise witness but loses:

```text
multiplicity of the response
position of the response
negative evidence below the maximum
co-occurrence of channel responses
```

Two trajectories can collide:

```math
\max_t[y_t]_r
=
\max_t[y_t']_r
\qquad
\forall r.
```

A sparse positive target can benefit from max, while a burden or prevalence
target requires a statistic that retains multiplicity.

## 14. Attention readout failure

The released MambaMIL head scores contextualized tokens:

```math
s_t
=
w_2^{\mathsf T}
\tanh(W_1h_t+b_1)+b_2,
```

```math
\alpha_t
=
\frac{\exp(s_t)}{\sum_u\exp(s_u)},
\qquad
z
=
\sum_t\alpha_th_t.
```

The coefficient is a normalized representation weight. It is not a patch
posterior:

```math
\alpha_t
\ne
\Pr(Z_t=1\mid Y,\mathcal B)
```

without a separately specified probabilistic model and calibration argument.

For a scalar task head q equals w-transpose-z plus b, the signed additive term is:

```math
r_t
=
\alpha_t w^{\mathsf T}h_t.
```

The full input-patch derivative includes the scan and score paths:

```math
\frac{\partial q}{\partial u_k}
=
w^{\mathsf T}
\sum_t
\left[
\frac{\partial\alpha_t}{\partial u_k}h_t
+
\alpha_t\frac{\partial h_t}{\partial u_k}
\right].
```

Because h-t depends on earlier inputs and alpha-t depends on all scores,
attention alone cannot identify the effect of patch k.

## 15. Attention duplication counterexample

Duplicate token j with identical state and score. The original denominator is:

```math
S
=
\sum_t\exp(s_t).
```

The duplicated denominator is:

```math
S'
=
S+\exp(s_j).
```

The two copies receive:

```math
\alpha_j'
=
\frac{\exp(s_j)}{S'},
\qquad
\alpha_{j'}'
=
\frac{\exp(s_j)}{S'}.
```

Their combined mass is 2 exp(s-j) divided by S-prime, while every other token
loses mass. The representation generally changes even though the duplicated
token value is identical:

```math
z'
=
\frac{
\sum_{t\ne j}\exp(s_t)h_t
+
2\exp(s_j)h_j
}{
S+\exp(s_j)
}.
```

Attention is therefore not invariant to multiplicity. This can be desirable
when repeated morphology represents burden, or harmful when patch sampling
density changes across slides.

## 16. Order perturbation versus token deletion

For a scan model, define order perturbation:

```math
\Delta_{\mathrm{order}}
=
q(P_\sigma H)-q(P_{\sigma'}H).
```

Define deletion:

```math
\Delta_{\mathrm{delete},k}
=
q(P_\sigma H)-q(P_\sigma H\text{ with token }k\text{ removed}).
```

These are different interventions. Deleting token k changes sequence length and
all later prefix states. Reordering keeps the multiset but changes every prefix
assignment.

A stable model should report both:

```math
\Delta_{\mathrm{order}}
\quad\text{and}\quad
\Delta_{\mathrm{delete},k}.
```

A small deletion effect does not prove that the token is irrelevant if its
information is duplicated elsewhere. A small order effect does not prove set
invariance if the tested orders are too similar.

## 17. Boundary and length effects

Let the final-state contribution of a token be:

```math
c_k
=
\overline A^{L-k}\overline B u_k.
```

The same patch feature placed at the beginning and end has different memory
multipliers:

```math
c_{\mathrm{first}}
=
\overline A^{L-1}\overline B u,
\qquad
c_{\mathrm{last}}
=
\overline B u.
```

For max or attention over all outputs the difference is more complicated, but
the scan boundary still changes the set of prefixes. A patch's importance can
therefore depend on its position even if its morphology is unchanged.

Length normalization can also change the effective representation. If two
slides contain the same local sequence followed by unrelated patches, then:

```math
x_L
=
\overline A^m x_{\mathrm{prefix}}
+
\text{suffix contribution}.
```

The prefix is attenuated or transformed by the suffix horizon. A final-state
head is not length-invariant.

## 18. Long-range interaction test

For token k and output t, define influence:

```math
I_{tk}
=
\left\|
\frac{\partial y_t}{\partial u_k}
\right\|_{\mathrm F}.
```

For a linear stable SSM:

```math
I_{tk}
\lesssim
C_0
\rho(\overline A)^{t-k}
\qquad
k\le t.
```

Estimate I-t-k across order distance and physical distance. A model can have
small sequence-distance decay but large physical-distance decay if the order
preserves local geometry, or the reverse if the order is unrelated to tissue
layout.

A useful two-axis diagnostic is:

```math
I_{\mathrm{seq}}(r)
=
\mathbb E
\left[
I_{tk}
\mid
|t-k|=r
\right],
```

```math
I_{\mathrm{phys}}(r)
=
\mathbb E
\left[
I_{tk}
\mid
\|p_t-p_k\|_2\in\text{bin }r
\right].
```

This distinguishes a long-memory claim from a spatial-context claim.

## 19. Task-head mismatch

The same SSM field can feed different tasks:

```math
\eta_i^{\mathrm{Cox}}
=
w_{\mathrm{Cox}}^{\mathsf T}z_i,
```

```math
\widehat p_i
=
\mathrm{softmax}(W_{\mathrm{cls}}z_i+b_{\mathrm{cls}}),
```

```math
\widehat h_i^{(k)}
=
\sigma(w_k^{\mathsf T}z_i+b_k).
```

Max may preserve a rare witness for classification but fail to preserve
prevalence. Attention can rank a token highly for a class head while a survival
head uses a different direction in the same z.

The appropriate deletion score is task-specific:

```math
\Delta_{ik}^{(q)}
=
q_i
-
q_i^{(-k)}.
```

A heatmap generated for q-class should not be presented as an explanation of
q-Cox without recomputing the latter.

## 20. Sampling and order leakage

If sequence order is derived from file order, patch extraction order, or a
precomputed top-k list, it can contain acquisition or preprocessing artifacts.
Let A be an acquisition variable and Y the label. The order can create a
shortcut when:

```math
I(\sigma;Y\mid H)
>
0
```

because sigma contains information about A rather than tissue.

A split-safe construction requires:

```math
\sigma_{\mathrm{train}}
=
\Gamma_{\sigma}(P_{\mathrm{train}},H_{\mathrm{train}}),
\qquad
\sigma_{\mathrm{test}}
=
\Gamma_{\sigma}(P_{\mathrm{test}},H_{\mathrm{test}}).
```

If a learned ordering or normalization is fit on all slides before splitting,
the sequence geometry has leaked test distribution information.

Compare performance under:

```text
original order
random within-slide permutations
coordinate-sorted order
Hilbert or raster order
repeated random order ensemble
```

A sharp drop under order randomization demonstrates dependence on G; it does not
by itself show that the original order is biologically valid.

## 21. C/R/G/S matrix

| Method | C: context | R: readout | G: induced geometry | S: supervision | Primary collision |
| --- | --- | --- | --- | --- | --- |
| S4MIL | structured S4D sequence convolution | coordinatewise max | inherited patch/grid order | slide label plus optional patch loss | same channelwise extrema |
| MambaMIL | selective causal Mamba blocks with residuals | released-code attention weighted mean | original sequence order | classification or discrete survival head | same weighted state vector |
| BiMamba | forward and reverse selective scans | downstream task readout | two directions of chosen order | slide-level task loss | directional summaries can still collide |
| SR-Mamba | original and segment-transposed selective scans | fused branch readout | original plus segment geometry | slide-level task loss | same fused two-order trajectory |

The table must state whether it describes the paper or a released
implementation. In particular, a survival hazard head is not implied by Mamba;
it is a task-head choice.

## 22. State-space interventions

### 22.1 Permutation test

Apply several permutations while preserving the patch multiset:

```math
\Delta_{\sigma,\sigma'}
=
d
\left(
F(P_\sigma H),
F(P_{\sigma'}H)
\right).
```

Report its distribution, not one convenient pair.

### 22.2 Scan-direction test

Compare forward and reverse scans:

```math
\Delta_{\mathrm{dir}}
=
d
\left(
F_{\rightarrow}(H),
F_{\leftarrow}(H)
\right).
```

A large value reveals directional dependence.

### 22.3 Reordering test

For SR-Mamba, ablate each branch:

```math
F_{\mathrm{orig}},
\qquad
F_{\mathrm{reord}},
\qquad
F_{\mathrm{fused}}.
```

This isolates the contribution of the second geometry.

### 22.4 Padding test

Append inert padding and verify:

```math
F(H)
=
F(H\Vert 0_{\mathrm{pad}})
```

under a correct mask and de-padding convention.

### 22.5 Readout swap

Hold Y fixed and compare:

```math
z^{\mathrm{max}},
\qquad
z^{\mathrm{mean}},
\qquad
z^{\mathrm{attn}},
\qquad
z^{\mathrm{final}}.
```

This isolates R from C.

### 22.6 Prefix deletion

Remove a token and recompute every later state. Do not use a fixed-state
attention approximation as the only deletion test.

## 23. C/R/G/S reporting rule

A state-space MIL paper is minimally specified when it reports:

```text
1. the initial patch object and exact sequence construction
2. the order source and whether it is spatial, learned, or arbitrary
3. state dimension, channel width, and discretization or selective parameterization
4. causal, bidirectional, or multi-order scan directions
5. padding, masking, and de-padding conventions
6. residual, convolution, gating, and normalization paths
7. the exact readout after the scan
8. whether readout scores act on raw, contextualized, or fused states
9. task head and likelihood or loss
10. order, direction, length, and readout ablations
```

The four-axis summary is:

```math
\begin{aligned}
G &: \text{sequence order and optional physical coordinates},
\\
C &: \text{finite-state recurrent or convolutional scan},
\\
R &: \text{max, mean, attention, final state, or fusion},
\\
S &: \text{slide, patch, survival, and auxiliary supervision}.
\end{aligned}
```

## 24. Bottom line

State-space MIL is not set MIL with a faster pooling layer. Its map is:

```math
\boxed{
\text{patch multiset}
\longrightarrow
\text{ordered sequence}
\longrightarrow
\text{finite selective trajectory}
\longrightarrow
\text{surviving readout statistic}
\longrightarrow
\text{task output}
}
```

The central mathematical risks are:

```text
order is an unacknowledged geometry
finite state creates prefix collisions
stable modes attenuate long-range influence
selective gates make influence input-dependent
padding can enter both state and readout
max and attention create different collision classes
task supervision can reward order or acquisition shortcuts
```

The correct claim is therefore:

```math
\boxed{
\text{state-space efficiency changes the context operator, but it does not
remove the need to specify order, readout, surviving statistic, and supervision}
}
```

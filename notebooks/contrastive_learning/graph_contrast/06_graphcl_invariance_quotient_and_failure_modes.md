# GraphCL Invariance Quotient And Failure Modes

Primary anchor:

- You et al. "Graph Contrastive Learning with Augmentations." NeurIPS 2020.
  https://arxiv.org/abs/2010.13902

## Augmentation Reachability

Let `\mathcal{T}` be the support of graph transformations. Define one-step
reachability:

```math
G\leadsto G'
\quad\Longleftrightarrow\quad
q(G'\mid G)>0.
```

Let `\sim_{\mathcal{T}}` be the equivalence relation generated by symmetric,
transitive closure of reachability. Ideal perfect alignment implies:

```math
z(G)
=
z(G')
\qquad
\text{whenever}
\qquad
G\sim_{\mathcal{T}}G'.
```

The encoder factors through a quotient:

```math
\mathcal{G}
\longrightarrow
\mathcal{G}/\!\sim_{\mathcal{T}}
\longrightarrow
\mathbb{R}^{d}.
```

Any target varying inside an augmentation orbit becomes unrecoverable from a
perfectly invariant representation.

## Destructive Invariance

Let target `Y` and view `\widetilde G` satisfy:

```math
I(Y;\widetilde G)
<
I(Y;G).
```

By data processing, a representation of the view obeys:

```math
I
\left(
Y;z(\widetilde G)
\right)
\le
I(Y;\widetilde G).
```

Contrastive alignment cannot recover task information erased by augmentation.

## Topology-Label Collision

Suppose two classes differ only by edge `e`:

```math
Y(G)=0,
\qquad
Y(G\cup\{e\})=1.
```

If edge perturbation assigns positive probability to adding or deleting `e`,
then both class states lie in the same augmentation orbit. Perfect invariance
requires:

```math
z(G)
=
z(G\cup\{e\}),
```

which makes `Y` non-identifiable from `z`.

## Subgraph Ambiguity

Two source graphs may generate the same subgraph `S`:

```math
q(S\mid G_a)>0,
\qquad
q(S\mid G_b)>0,
\qquad
Y_a\ne Y_b.
```

No deterministic encoder can infer which source generated `S` from the view
alone. The positive-pair task contains irreducible ambiguity.

## Augmentation Difficulty Is Not Semantic Validity

Let contrastive Bayes error under kernels `q_1,q_2` be:

```math
\mathcal{E}_{\mathrm{ctr}}^{\star}
\left(
q_1,q_2
\right).
```

Harder augmentations may increase this error, but downstream validity depends
on:

```math
\Delta_Y(q)
=
I(Y;G)-I(Y;\widetilde G).
```

There is no general monotone relation between contrastive difficulty and
`\Delta_Y`. A hard task can be useful, impossible, or destructive.

## WSI Counterexample

Let slide label be positive exactly when one tumor node exists:

```math
Y(G)
=
\mathbb{1}
\left\{
\exists i:
T_i=1
\right\}.
```

If node dropping removes that unique node, the augmented graph has the same
observed state as a truly negative slide:

```math
\widetilde G_{+}
=
G_{-}.
```

The positive pair asks the encoder to align `G_+` with `G_-` while other-slide
negatives may ask it to separate morphologically identical benign graphs.
This is a supervision contradiction induced by sparse positives.

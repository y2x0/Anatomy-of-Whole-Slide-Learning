# Graph Contrast

This portion asks:

```text
Which graph object is declared invariant, and which alternatives define its
contrastive geometry?
```

Only three fully accessible papers already listed in the private research map
are used: DGI, GraphCL, and GRACE. The inaccessible pathology hypergraph paper
is intentionally excluded.

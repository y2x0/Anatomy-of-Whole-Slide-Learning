# Unifying View

Pooling operators are readout operators in the repository-wide decomposition:

```math
\widetilde H_i
=
\mathcal{C}(H_i;G_i,S_i),
\qquad
z_i
=
\mathcal{R}(\widetilde H_i;G_i,S_i),
\qquad
\widehat y_i
=
\mathcal{H}(z_i).
```

This folder keeps the pooling family aligned with the C/R/G/S framework.

## Files

- `01_crgs_pooling_decomposition.md`: how C, R, G, and S specialize for pooling.
- `02_paper_placement_matrix.md`: mean, max, ABMIL, DSMIL, CLAM, and PANTHER.

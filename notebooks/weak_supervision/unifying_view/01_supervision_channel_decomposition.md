# Supervision Channel Decomposition

Every weak-supervision method can be decomposed into four objects:

```text
latent truth U
observed supervision S^obs
generated target U_hat_t
model prediction F_theta
training loss L
```

The statistical channel is:

```math
U
\xrightarrow{Q}
S^{\mathrm{obs}}.
```

The learning channel is:

```math
(H,G,S^{\mathrm{obs}})
\xrightarrow{\mathcal{L}}
\theta.
```

Pseudo-label and teacher-student methods add an algorithmic target:

```math
\widehat U_t
=
\Psi_t(H,G,S^{\mathrm{obs}},\theta_t,\mathcal{D}).
```

This target is endogenous. It is not sampled from the annotation channel
```math
Q
```
.

## Latent Truth

For WSI learning:

```math
U_i
=
\left(
Y_i,
Z_i,
B_i,
A_i,
R_i
\right).
```

where:

```text
Y_i:
    true slide target

Z_i:
    latent patch or region labels

B_i:
    burden or prevalence object

A_i:
    latent assignment, attention, or evidence variable

R_i:
    report, region, or annotation object
```

## Observed Signal

The observed signal can be:

```math
S_i^{\mathrm{obs}}
\in
\{
Y_i,
\widetilde Y_i,
M_i\odot Z_i,
\mathcal{P}_i,
T_i
\}.
```

These correspond to:

```text
bag label
noisy label
partial label
contrastive relation
text or report pair
```

Generated targets include:

```math
\widehat U_{i,t}
\in
\{
\widehat Z_{ij,t},
\widehat Y_{im,t},
q_{\phi,t}(U_i\mid H_i,G_i),
T_i^{+},
T_i^{-},
\mathrm{cluster}_{ij,t}
\}.
```

## Surrogate Loss

The loss is:

```math
\mathcal{L}_{\mathrm{sur}}(\theta)
=
\sum_i
\ell
\left(
F_\theta(H_i,G_i),
S_i^{\mathrm{obs}}
\right).
```

The desired target loss is:

```math
\mathcal{L}_{\mathrm{target}}(\theta)
=
\sum_i
L
\left(
F_\theta(H_i,G_i),
U_i
\right).
```

Do not define the theory by subtracting these two losses. They have different
arguments and scales. The useful object is excess target risk:

```math
\mathcal{E}_{\mathrm{target}}(\theta)
=
\mathcal{L}_{\mathrm{target}}(\theta)
-
\inf_{\theta'}\mathcal{L}_{\mathrm{target}}(\theta').
```

The surrogate is useful when:

```math
\mathcal{E}_{\mathrm{target}}(\theta)
\le
\psi
\left(
\mathcal{L}_{\mathrm{sur}}(\theta)
-
\inf_{\theta'}\mathcal{L}_{\mathrm{sur}}(\theta')
\right)
```

for a nondecreasing calibration function
```math
\psi
```
under stated assumptions.

## Channel Types

Bag label:

```math
S^{\mathrm{obs}}
=
\Gamma(Z).
```

Noisy label:

```math
S^{\mathrm{obs}}
\sim
T(\cdot\mid Y).
```

Partial label:

```math
S^{\mathrm{obs}}
=
(M,M\odot U).
```

Pseudo-label:

```math
\widehat U_t
=
\Psi_t(H,G,S^{\mathrm{obs}},\theta_t,\mathcal{D}).
```

This is generated supervision, not an exogenous observation channel.

Contrastive label:

```math
S^{\mathrm{obs}}
=
\{(a,b):U_a\sim U_b\}.
```

## Dense Summary

The core decomposition is:

```math
U
\to
S^{\mathrm{obs}}
\to
\mathcal{L}_{\mathrm{weak}}.
```

A method is mathematically clear when it states whether the target is observed
through
```math
Q
```
 or generated by
```math
\Psi_t
```
.

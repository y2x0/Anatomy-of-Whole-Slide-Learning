# Problem Setup

Weak supervision in WSI learning is a latent-variable problem.

The ideal fully supervised dataset would contain:

```math
\mathcal{D}_{\mathrm{full}}
=
\{(H_i,G_i,U_i)\}_{i=1}^{N},
```

where $U_i$ contains all task-relevant truth:

```math
U_i
=
\left(
Y_i,
Z_{i1},\ldots,Z_{in_i},
B_i,
R_i
\right).
```

Here:

```text
Y_i:
    true slide-level target

Z_ij:
    true instance-level target

B_i:
    true burden, count, prevalence, or risk object

R_i:
    true region-level or report-level object
```

The actual weakly supervised dataset contains observed supervision:

```math
\mathcal{D}_{\mathrm{weak}}
=
\{(H_i,G_i,S_i^{\mathrm{obs}})\}_{i=1}^{N},
```

where $S_i^{\mathrm{obs}}$ is observed supervision, not complete truth.

## Observation Channel

The observed signal is generated by:

```math
S_i^{\mathrm{obs}}
\sim
Q_\alpha(S\mid U_i,H_i,G_i).
```

The learning problem is not only to estimate a prediction function:

```math
f_\theta(H_i,G_i)
\approx
Y_i.
```

It is to do so while only observing $S_i^{\mathrm{obs}}$:

```math
S_i^{\mathrm{obs}}
\not=
U_i.
```

This is the exogenous observation channel. Pseudo-labels, teacher targets,
attention-selected sets, and pseudo-bag labels are not new observations from
this channel. They are generated by the training algorithm:

```math
\widehat U_{i,t}
=
\Psi_t(H_i,G_i,S_i^{\mathrm{obs}},\theta_t,\mathcal{D}).
```

This channel view covers:

```text
bag labels:
    S_i^obs = Y_i, patch labels Z_ij hidden

partial labels:
    S_i^obs contains labels for a subset of patches or regions

noisy labels:
    S_i^obs is a corrupted version of Y_i or Z_ij

pseudo-labels:
    S_i^obs is the original weak label; U_hat_i,t is generated by the model or teacher

contrastive labels:
    S_i^obs specifies which pairs or views should be similar
```

## Latent Instance Model

For binary MIL, let:

```math
Z_{ij}
\in
\{0,1\}.
```

The classic positive-instance assumption is:

```math
Y_i
=
\max_{1\le j\le n_i}Z_{ij}.
```

Equivalently:

```math
Y_i=1
\quad\Longleftrightarrow\quad
\exists j:Z_{ij}=1.
```

The negative bag constraint is strong:

```math
Y_i=0
\quad\Longrightarrow\quad
Z_{ij}=0
\quad
\forall j.
```

The positive bag constraint is weak:

```math
Y_i=1
\quad\Longrightarrow\quad
\sum_{j=1}^{n_i}Z_{ij}\ge 1.
```

It does not say which patch is positive, how many are positive, or whether the
positive patches are sufficient evidence by themselves.

## Marginal Likelihood

A probabilistic MIL model introduces instance probabilities:

```math
p_{ij}
=
P_\theta(Z_{ij}=1\mid h_{ij}).
```

Under conditional independence and the OR rule:

```math
P_\theta(Y_i=1\mid H_i)
=
1-\prod_{j=1}^{n_i}(1-p_{ij}).
```

The bag-label log likelihood is:

```math
\ell_i(\theta)
=
y_i\log
\left(
1-\prod_{j=1}^{n_i}(1-p_{ij})
\right)
+
(1-y_i)
\sum_{j=1}^{n_i}\log(1-p_{ij}).
```

This is a likelihood for bags, not direct supervision for patch labels.

## Empirical Risk Under Weak Labels

Most WSI methods optimize a weak surrogate:

```math
\widehat{\mathcal{R}}_{\mathrm{sur}}(\theta)
=
\frac{1}{N}
\sum_{i=1}^{N}
\ell
\left(
F_\theta(H_i,G_i),
S_i^{\mathrm{obs}}
\right).
```

The target risk is:

```math
\mathcal{R}_{\mathrm{target}}(\theta)
=
\mathbb{E}
\left[
L(F_\theta(H,G),U)
\right].
```

Weak supervision is statistically valid only when the surrogate controls the
target in a calibration or excess-risk sense. A useful statement has the form:

```math
\mathcal{R}_{\mathrm{target}}(\theta)
-
\mathcal{R}_{\mathrm{target}}^\star
\le
\psi
\left(
\mathcal{R}_{\mathrm{sur}}(\theta)
-
\mathcal{R}_{\mathrm{sur}}^\star
\right),
```

where $\psi$ is nondecreasing and the statement holds only under explicit
assumptions about the supervision channel. Directly subtracting weak and target
losses is usually meaningless because they live on different observation spaces.

## Observed Likelihood With A Channel

If:

```math
P_\theta(U\mid H,G)
```

models latent pathology truth and:

```math
Q_\alpha(S^{\mathrm{obs}}\mid U,H,G)
```

models how truth becomes observed supervision, then:

```math
P_{\theta,\alpha}(S^{\mathrm{obs}}\mid H,G)
=
\sum_U
Q_\alpha(S^{\mathrm{obs}}\mid U,H,G)
P_\theta(U\mid H,G).
```

For continuous or structured $U$, the sum is replaced by an integral. This keeps
the pathology model and the supervision channel separate.

## Identifiability

Patch labels are not identifiable from bag labels without assumptions.

There can be two instance classifiers:

```math
p_\theta(z_j=1\mid h_j)
\ne
p_{\theta'}(z_j=1\mid h_j)
```

that induce the same bag probability:

```math
P_\theta(Y\mid H)
=
P_{\theta'}(Y\mid H).
```

If the observed objective only sees $Y$, it cannot distinguish these patch-level
explanations.

This is the source of attention-map ambiguity, pseudo-label drift, and
instance-level overinterpretation.

## C/R/G/S Placement

Supervision can enter the WSI pipeline at several points:

```text
S in C:
    labels or generated targets shape instance features before readout

S in R:
    labels or generated targets induce attention weights, top-k sets, prototypes, or bag constraints

S in G:
    labels shape graphs, regions, neighborhoods, or learned topology

S in H:
    labels supervise the final task head only
```

The same architecture can behave differently under different supervision
channels.

## Dense Summary

Weak supervision is:

```math
U
\to
S^{\mathrm{obs}}
\to
\mathcal{L}_{\mathrm{weak}}
\to
\theta,
```

where $U$ is the unobserved pathology truth and $S^{\mathrm{obs}}$ is the
observed training signal.

Pseudo-labeling adds:

```math
S^{\mathrm{obs}},\theta_t,\mathcal{D}
\to
\widehat U_t
\to
\mathcal{L}_{\mathrm{generated}}.
```

The most important question is not:

```text
What labels do we have?
```

It is:

```text
What latent variables do those labels actually constrain?
```

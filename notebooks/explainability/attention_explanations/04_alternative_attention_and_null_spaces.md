# Alternative Attention And Null Spaces

Primary anchors:

- Jain, Wallace. "Attention is not Explanation." NAACL 2019.
  https://arxiv.org/abs/1902.10186
- Wiegreffe, Pinter. "Attention is not not Explanation." EMNLP 2019.

## Fixed-Value Equivalence

Stack value vectors as columns:

```math
V
=
\left[
v_1,\ldots,v_n
\right]
\in
\mathbb{R}^{d\times n}.
```

The attention readout is:

```math
z
=
V\alpha.
```

An alternative distribution `alpha'` preserves the readout when:

```math
V
\left(
\alpha'-\alpha
\right)
=
0,
```

```math
\mathbf{1}^{\top}
\left(
\alpha'-\alpha
\right)
=
0,
```

and:

```math
\alpha'
\ge
0.
```

## Constrained Null Space

Let:

```math
\mathcal{N}
=
\mathrm{null}(V)
\cap
\mathrm{null}
\left(
\mathbf{1}^{\top}
\right).
```

Any sufficiently small `delta` in this space yields:

```math
\alpha'
=
\alpha+\delta
```

with identical readout.

## Dimension Lower Bound

Stack constraints:

```math
\widetilde V
=
\begin{bmatrix}
V
\\
\mathbf{1}^{\top}
\end{bmatrix}
\in
\mathbb{R}^{(d+1)\times n}.
```

Then:

```math
\dim
\mathrm{null}
\left(
\widetilde V
\right)
=
n-
\mathrm{rank}
\left(
\widetilde V
\right)
\ge
n-d-1.
```

For WSI bags with `n` far larger than embedding dimension `d`, many
prediction-preserving attention directions can exist.

## Logit-Preserving Alternatives

If only one linear logit matters, alternatives need satisfy the weaker
constraint:

```math
w_c^{\top}V
\left(
\alpha'-\alpha
\right)
=
0.
```

Its null space is generally even larger than the readout-preserving space.

## Distance Maximization

An adversarial attention search can solve:

```math
\max_{\alpha'\in\Delta^{n-1}}
D
\left(
\alpha',\alpha
\right)
```

subject to:

```math
\left|
F(V\alpha')-F(V\alpha)
\right|
\le
\varepsilon.
```

Large feasible distance demonstrates nonuniqueness under that intervention
protocol. It does not prove the trained attention mechanism itself is useless.

## Primitive Versus Mechanism

Replacing attention weights after training treats the distribution as a free
primitive. In the trained model, attention is generated by parameters and
inputs. Model-level diagnostics must also ask whether the learned mechanism
contributes to predictive performance and generalization.

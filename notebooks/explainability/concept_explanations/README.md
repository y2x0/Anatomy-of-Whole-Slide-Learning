# Concept Explanations

This portion asks:

```text
When does a human-named or automatically discovered direction explain a model?
```

A concept can be a supervised separating direction, an automatically clustered
visual pattern, a sparse latent coordinate, or a language-derived pathology
prior. The explanation may report directional sensitivity, population
frequency, concept presence, score credit, or predictive change. These claims
require different mathematics and different validation.
